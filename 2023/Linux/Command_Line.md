#### cd

The `cd` command is used to change working directory

| Command | Result                  |
| ------- | -------                 |
| `cd`    | Changes to home directory        |
| `cd usr/bin`    | Change to `usr/bin` directory       |
| `cd ..`    | Changes to parent directory       |
| `cd - `    | Changes to previous directory       |
